Let's Learn Pipes

lowercase, uppercase, titlecase Pipe::

UserName without pipe:: {{userName}}
UserName with uppercase pipe:: {{userName | uppercase}}
UserName with lowercase pipe:: {{userName | lowercase}}
UserName with titlecase pipe:: {{userName | titlecase}}

Currency Pipe::

{{mySal}}
{{mySal | currency}}
{{ mySal | currency : 'USD' : 'code' }}
{{ mySal | currency : 'USD' : 'symbol' }}
{{mySal | currency : 'INR'}}
{{mySal | currency : '€'}}
{{mySal | currency : '元'}}
